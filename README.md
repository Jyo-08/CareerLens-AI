# CareerLens AI

CareerLens AI is an intelligent career guidance platform that helps students and job seekers understand how recruiters evaluate their resumes. The platform goes beyond traditional ATS scoring by providing career readiness insights, recruiter-style feedback, personalized growth recommendations, and structured career roadmaps.

## Overview

Many candidates apply for internships and jobs without a clear understanding of how their resumes are perceived by recruiters. Existing resume analyzers often provide limited feedback focused only on ATS compatibility.

CareerLens AI transforms a resume into a comprehensive career intelligence report by analyzing strengths, identifying skill gaps, evaluating role compatibility, and generating actionable recommendations for career growth.

## Problem Statement

Students and early-career professionals often struggle to answer critical questions:

* Is my resume recruiter-ready?
* What skills am I missing for my target role?
* How competitive am I compared to industry expectations?
* What improvements will have the highest impact on my career growth?

Most available solutions provide generic ATS scores without meaningful guidance.

## Solution

CareerLens AI provides a structured career analysis workflow that helps users understand their current standing and identify the most effective next steps for professional growth.

The platform focuses on:

* Resume evaluation
* ATS compatibility analysis
* Career readiness assessment
* Skill gap identification
* Recruiter-style feedback
* Personalized career roadmaps
* Progress tracking

## Features

### Resume Analysis

Upload a resume and receive detailed career insights based on role-specific expectations.

### ATS Compatibility Assessment

Evaluate resume quality and identify areas that may impact applicant tracking system performance.

### Career Readiness Level

Understand your current level of career preparedness and what is required to progress further.

### Skill Gap Analysis

Identify missing skills and competencies relevant to your target role.

### Recruiter AI

Interact with an AI-powered recruiter assistant to receive personalized guidance and recommendations.

### Personalized Career Roadmap

Receive a structured action plan designed to improve employability and career readiness.

### Career Progress Tracking

Track improvements across multiple resume analyses and monitor growth over time.

### User Authentication

Supports account-based access to enable a personalized and private user experience.

## Privacy and Security

CareerLens AI follows a privacy-focused design philosophy.

Future implementations will ensure that user data remains isolated and accessible only to authenticated users. Resume processing, analysis storage, and career history management will be designed with user privacy and security as core priorities.

## Installation

Clone the repository:

```bash
git clone https://github.com/Jyo-08/CareerLens-AI.git
```

Navigate to the project directory:

```bash
cd CareerLens-AI
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
